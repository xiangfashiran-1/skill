# 异常处理规则

## 发送失败

- 提示：日报发送失败，渠道配置可能异常。修复后可输入「重发今日日报」重新发送。
- 自动检查：配置文件是否完整、渠道凭证是否过期
- 引导用户重新校验配置

## 配置缺失

- 用户尝试使用日报功能但未完成配置时，自动触发首次配置引导
- 提示：检测到日报功能尚未配置，现在帮你完成配置～

## 记录文件不存在

- 当日无工作记录时生成日报：提示"今日暂无工作记录，是否手动补充？"
- 不生成空日报

## 临时改时间

- 仅当日生效，次日自动恢复默认时间
- 提示：今日发送时间已临时修改，仅当日生效，明日自动恢复默认时间。

## 周末/节假日

- 周末（周六、周日）：今日是周末，不生成日报，祝你休息愉快～
- 法定节假日：今日是法定节假日，日报功能暂停，祝假期愉快～
- 判断逻辑：先检查星期几，再检查是否为中国法定节假日（可通过 web 查询确认）

## 重复配置

- 用户已配置完成后再次触发配置引导：提示当前配置摘要，询问是否重新配置
